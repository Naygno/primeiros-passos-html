A linguagem de marcação HTML tem por objetivo criar a estrutura de uma página Web. Ela é como se fosse as partes fundamentais de uma casa, como teto, paredes e piso, mas sem o acabamento e demais instalações como água e energia elétrica. Para isso ela utiliza elementos que são a combinação de uma tag de abertura, seus atributos, o conteúdo que está entre as tags e a tag de fechamento.

Estrutura Base do Documento

Esses são os elementos e a declaração que formam a estrutura esquelética de toda página HTML. As principais tags são:

  1. <!DOCTYPE html>
  2. <html>
  3. <head>
  4. <body>

Veja as explicações e usos dessas tags a seguir.

1. <!DOCTYPE html>

Não é propriamente uma tag, mas uma declaração que informa ao navegador qual versão do HTML o documento está usando. Deve ser a primeira linha do código.

2. <html>

Envolve todo o conteúdo da página web. É o elemento raiz.

3. <head>

Nele devem ser incluídos elementos como título que será mostrado na aba do navegador e meta elementos, que não serão mostrados na página, para estilização e funcionamento da página como endereços para arquivos CSS e JavaScript.

4. <body>

Essa tag deve envolver todo o conteúdo que será mostrado no navegador ao usuário.

Estrutura de Seção e Conteúdo

Esses elementos definem a organização do conteúdo criando divisões e hierarquias.

  1. <header>
  2. <h1> até <h6>
  3. <p>
  4. <blockquote>
  5. <hr>

Explicações e usos dessas tags.

1. <header>

É onde fica o conteúdo introdutório ou grupo de links de navegação pela página.

2. <h1> até <h6>

Essas tags são importantes para a criação de títulos para as seções e indicar a sua hierarquia.

  • A tag <h1> é a mais importante na hierarquia e deve haver apenas um por página.
  • A tag <h2> é a segunda na ordem de importância de deve seguir sempre depois de <h1>
  • As demais tags <h3>, <h4>, <h5> e <h6> vêm em seguida respeitando sua ordem na hierarquia.

3. <p>

Essa tag delimita os parágrafos de textos na página.

4. <blockquote>

É usada para criar um bloco de texto de outra fonte de origem.

5. <hr>

Cria uma linha horizontal que delimita seções em uma página.

Listas

Essas tags trabalham em conjunto para organizar itens em forma de listas.
  1. <ul>
  2. <ol>
  3. <li>

Explicações e usos dessas tags.

1. <ul>

Cria uma lista não ordenada (com marcadores).

2. <ol>

Cria uma lista ordenada (com numeração).

3. <li>

Cria um item em uma lista.

Ênfase e Semântica de Texto

Essas tags atribuem um significado ou ênfase a partes específicas do texto.

  1. <strong>
  2. <i>
  3. <small>
  4. <mark>
  5. <abbr>
  6. <del>

Explicações e usos dessas tags.

1. <strong>

Indica que o texto tem grande importância, urgência ou seriedade. O texto fica em negrito.

2. <i>

Represente um texto em um tom ou voz diferente. O texto fica em itálico.

3. <small>

Usado para texto com informações secundárias, como avisos ou direitos autorais. O texto fica menor.

4. <mark>

Destaca um trecho de texto por sua relevância em um contexto específico. O texto fica destacado como os marcadores de texto fazem.

5. <abbr>

Define a abreviação ou sigla. As iniciais HTML é um abreviação para HyperText Markup Language. Esse elemento deve vir acompanhado do atributo title com a descrição completa da abreviação ou sigla contida como valor. Esse atributo faz o navegador responder mostrando um balãozinho com o significado da sigla ou abreviação quando o cursor do mause passa sobre ela.

6. <del>

Indica que o texto foi removido de um documento. Este texto foi removido.

Formatação de Texto

Essas tags modificam a aparência do texto. Mas isso também pode ser feito pelo CSS que é a forma mais recomendada.

  1. <b>
  2. <u>
  3. <sub>
  4. <sup>
  5. <font>

Explicações e usos dessas tags.

1. <b>

Torna o texto em negrito sem adicionar semântica, usado apenas para estilo.Este texto está em negrito.

2. <b>

Sublinha o texto. Texto sublinhado.

3. <sub>

O texto fica subscrito, como em fórmulas químicas. O texto a seguir está subscrito.

4. <sup>

O texto fica sobrescrito, como em fórmulas matemáticas. O texto a seguir está sobrescrito.

5. <font>

Era utilizada para definir estilo de texto, mas foi completamente substituída por CSS. Portanto está obsoleta.
